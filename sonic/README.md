# SONiC

This directory contains resources specific to SONiC (Software for Open Networking in the Cloud).

## Structure

```
sonic/
└── configs/    # SONiC configuration templates and examples
```

## Overview

SONiC is an open-source network operating system. Resources here support integration between SONiC devices and the AI agents defined in this repository.

## Subdirectories

- **[configs/](configs/README.md)** – Configuration templates and examples for SONiC deployments.
