# Agents

This directory contains AI agent definitions, configurations, and orchestration logic.

## Structure

```
agents/
└── skills/    # Reusable skills that can be composed into agents
```

## Overview

Agents are autonomous units that combine skills and configuration to perform tasks related to SONiC networking and operations.

## Subdirectories

- **[skills/](skills/README.md)** – Reusable building blocks that agents can invoke.
